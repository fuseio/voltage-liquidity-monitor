# Voltage Liquidity Monitor

## Getting Started

1. Copy example env file:
  ```shell
  cp .env.example .env
  ```
2. Update the environment variables
3. Install the packages:
  ```shell
  npm install
  ```
4. Run the server:
  ```shell
  npm run dev
  ```

## Docker
1. Use
 ```docker-compose up```
 to setup and run in Docker Environment
